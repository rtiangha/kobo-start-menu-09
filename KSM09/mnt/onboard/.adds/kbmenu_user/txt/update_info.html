Warning:
  • Do not install an update package when the battery charge is low.
  • KSM does not check whether there is sufficent free disk space for the update. If you have some doubts, do not install the package.
  • KR/K install uncontrolled: This will install the complete Kobo update package that is available in .kobo. Depending on the content of the update package, this might disable KSM.
  • KR/K install controlled: This will install the Kobo update package that is available in .kobo, with the exception of those parts that are likely to disable KSM.
  • Rule of thumb:
    • Use KR install controlled for Kobo firmware update files.
    • Use KR install uncontrolled for patch packages by GeoffR, and packages specifically provided for KSM.
  • If you are unsure what to do, select: "give a recommendation." Depending on the size of the update package, this might need some minutes. No reason to get worried.
